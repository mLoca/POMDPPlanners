# POMDPPlanners Documentation Index

This file provides an index of all documentation files created for the POMDPPlanners package, organized by purpose.

---

## 📊 Visual Documentation

### Architecture Diagrams

1. **`pomdp_architecture_diagram.png` / `.pdf`**
   - Complete package architecture showing all layers
   - 5 layers: Core Abstractions → Implementations → Simulation → Data → Utilities
   - Shows inheritance, composition, usage, and creation relationships
   - Generated by: `architecture_diagram.py`

2. **`pomdp_interaction_flow.png` / `.pdf`**
   - Sequence diagram of runtime interactions
   - Episode execution with MCTS planning
   - Shows method calls between components during execution
   - Generated by: `interaction_flow_diagram.py`

3. **`core_interface_relations.png` / `.pdf`** ⭐
   - Detailed graph of core module interfaces
   - Shows Environment, Policy, Belief, Distribution, Tree relationships
   - Includes inheritance, composition, and usage patterns
   - Generated by: `core_interface_graph.py`

## 🔄 Updating Documentation

### Regenerating Diagrams

If you modify the code and need to update diagrams:

```bash
# Ensure virtual environment is activated
source .venv/bin/activate

# Regenerate all diagrams
python architecture_diagram.py
python interaction_flow_diagram.py
python core_interface_graph.py
```

Each script generates both PNG (for viewing) and PDF (for papers) versions.
